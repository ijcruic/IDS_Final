Accountability: About This Data Set

This data set consists of Google activity recognition data and location data taken from a Samsung Android phone using the AWARE Framework. Data was collected from 2/17/2018 to 3/4/2018. This data is only drawn from my personally owened smart phone and the smart phone was only used by during the period of data collection, and so it only reflects information about me. During the time period of collection, I was also on vacation in Chicago, so some of the data reflects me on vacation, as opposed to a daily routine.

Where did this data come from?

This data orignated from a Samsung smart phone and was collected via the AWARE Framework. All of the data is from one cell phone solely operated by me, and so all data originates with my actions.

Who has access to it?

I am the sole curator of this data; I am the only person that updates, alters, deletes or otherwise manipulates the base data. As part of the Tableau display, a copy of the cleaned data used to make the display is also available on Tableau's public server. However, changes to the Tableau data will not reflect in the original data source.
